# Library Book Management System Using Stack

## Overview
This project simulates the borrowing and returning of books using a stack data structure implemented in Python. The system models book checkout operations using Last-In-First-Out (LIFO) logic to reflect real-world borrowing scenarios.

This program dynamically manages the inventory of available and borrowed books using object-oriented programming concepts and modular function design.

## Features
- Simulates borrowing and returning of books  
- Uses stack data structure for inventory tracking  
- Maintains updated inventory in real time  
- Implements LIFO-based checkout logic
